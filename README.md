\# URI (Unshackled Research Interface)



Local web UI + file-based RRR cycle watcher for orchestrating Ollama models.



\## Requirements

\- Windows PowerShell

\- Python 3.x

\- Ollama running at http://localhost:11434

\- Models installed (example):

&nbsp; - Reasoner: dolphin-llama3:latest

&nbsp; - Reducer: deepseek-r1:14b
